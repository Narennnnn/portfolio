my story

My first professional role was at Tonnochy Capital, a Hong Kong-based high-frequency trading startup. There, I built high-performance integrations for cryptocurrency market connectivity — working with REST APIs, WebSockets, and concurrency patterns to handle real-time data at scale. It was a fast-paced environment where milliseconds mattered, and I learned the value of clean, efficient, and fault-tolerant systems.

I’ve integrated Apple & Google Wallet Passes for the Planet Points program, built a .NET service for automated email notifications, migrated onboarding flows to passwordless login, and introduced personalized features like Instagram-style Stories. Every project has been an opportunity to combine technical precision with a focus on user experience.

Through these experiences, I’ve strengthened my skills in Java, Spring Boot, Python, Node.js, .NET, and modern JavaScript frameworks like Next.js. I’m drawn to building scalable, modular, and secure systems that handle complex data operations and real-time communication with ease.
